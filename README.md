# Citizen360

**Making Cities Better, One Complaint at a Time**

Citizen360 is a Smart City Initiative to make civic engagement simple, transparent, and effective. It allows citizens to report civic issues instantly and track their resolution with complete transparency. Powered by AI, GPS, and a network of dedicated officers, it connects citizens directly to the right municipal departments.

## 🌟 Features

- 📍 **GPS Location Detection:** Automatically pinpoint the exact location of any civic issue.
- 📸 **Upload Photos:** Attach evidence with multi-image uploads and instant preview.
- 🏢 **Auto Department Assignment:** Complaints routed to the correct department in seconds.
- 📊 **Live Complaint Tracking:** Follow every stage from submission to resolution.
- 🔔 **SMS & Email Alerts:** Never miss an update with real-time notifications.
- ✨ **AI Category Detection:** AI reads your photo and suggests the right category.
- 🗺️ **Interactive Map:** See what's happening in your neighborhood on a live map.

## 🏢 Departments Supported

- 🚧 **Roads:** Potholes, damage, signage
- 🗑️ **Sanitation:** Garbage & waste collection
- 💧 **Water Supply:** Leakage, shortage, quality
- ⚡ **Electricity:** Street lights, outages
- 🌊 **Drainage:** Blockages, flooding
- 🌳 **Parks & Gardens:** Maintenance, greenery

## 🚀 Tech Stack

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4, Radix UI Primitives
- **Routing:** Tanstack Router
- **Animations:** Framer Motion (motion/react)
- **Icons:** Lucide React
- **Data Fetching:** Tanstack React Query

## 🛠️ Getting Started

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/mvh2005/Citizen360.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Citizen360
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
